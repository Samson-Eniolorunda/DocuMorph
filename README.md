Repo/About text: DocuMorph is a privacy-first file tools web app that supports Convert, Compress, Resize, and Merge workflows with a clean upload → progress → download experience.

Tech stack: HTML, CSS, Vanilla JavaScript <br>
Processing: ConvertAPI <br>
Deployment: Vercel (secrets stored in environment variables) <br>

Features: 
<ul>
  <li>Responsive UI (desktop/mobile)</li>
  <li>Drag & drop uploads with validation</li>
  <li>Progress states + ready/success views</li>
  <li>Feedback & Requests modal (bug reports + tool requests) </li>
</ul>
